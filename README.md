# Projeto To-do list

## Objetivo
    Criar uma lista de afazeres diario.

## Funcionalidades
    1. Principais
        - [x] Criar um novo item
        - [x] Marcar o item como resolvido
        - [ ] Deletar item já criado
        - [ ] Alterar item já criado
        - [ ] Quando marcar um item como feito, passar uma linha sobre o item
